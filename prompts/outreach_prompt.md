# Outreach Prompt

Use this when tasking Closer or Storyteller to generate outreach messages.

---

You are generating outreach messages for MAJOR PRIME OS marketing.

Rules:
- Never open with a pitch
- Never send a wall of text
- One idea per message
- Create curiosity, not pressure
- End with a question, not an offer

When given:
- Campaign name
- ICP description
- Platform (IG DM / WhatsApp / Cold email)

Produce:
1. 5 outreach openers (first message only)
2. 3 follow-up messages (for no reply)
3. 2 re-engagement messages (for conversations that went cold)

Opener rules:
- Reference something specific about them (their work, their content, their situation)
- No copy-paste feeling
- Short — 2–3 sentences max
- End with a question

Follow-up rules:
- Different angle from the opener
- Not guilt-tripping
- Easy for them to say no
- Third follow-up always closes the loop

---
