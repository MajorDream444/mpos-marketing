# Content Calendar — MPOS Marketing

## Format

| Date | Campaign | Platform | Content Type | Hook / Topic | CTA | Status | Notes |
|------|----------|----------|--------------|--------------|-----|--------|-------|
| | | | | | | | |

---

## Status Options
- DRAFT — written, not reviewed
- READY — approved, ready to post
- SCHEDULED — set to go out
- LIVE — posted
- ARCHIVED — used, do not repost

---

## Weekly Template (Active Campaign)

### Monday
- Platform: Instagram
- Type: Reel
- Theme: Hook/problem-awareness
- CTA: DM me / Link in bio

### Tuesday
- Platform: WhatsApp Status + IG Story
- Type: Behind-the-scenes or social proof
- CTA: Reply / DM

### Wednesday
- Platform: Instagram Feed
- Type: Authority post (carousel or graphic)
- CTA: Save / Share / Comment

### Thursday
- Platform: IG Story
- Type: Poll or question — engagement
- CTA: Swipe up or DM

### Friday
- Platform: Instagram Reel
- Type: Results-focused or identity content
- CTA: Link in bio / Application

### Saturday
- Platform: WhatsApp Broadcast (campaign only)
- Type: Urgency / reminder

### Sunday
- Platform: IG Story
- Type: Personal or reflection
- CTA: Soft — "DM me if this resonates"

---

## Repurposing Rules
- Every reel → pull the hook as a caption post
- Every caption post → use the insight line as a WhatsApp status
- Every WhatsApp broadcast → repurpose as IG Story sequence
- Every DM that converts well → turn into a Storyteller template
