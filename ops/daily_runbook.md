# Daily Runbook — MPOS Marketing

## Purpose
Standardize daily execution so the marketing division runs on rhythm, not reaction.

## Morning Block (15–20 min)

### Step 1: Captain assigns
Run this command:
```
Run MPOS_MARKETING for today.

Campaign: [active campaign name]
Focus: [specific goal for today]

Assign tasks to:
- Scout: [find X leads / refresh Y list]
- Storyteller: [create X content pieces / write Y DMs]
- Distributor: [post X / schedule Y]
- Closer: [respond to X / send follow-ups to Y]
```

### Step 2: Review yesterday's results
- How many DMs sent?
- Any replies?
- Any applications?
- Any conversations to continue?

### Step 3: Check follow-up queue
- Anyone waiting on a reply?
- Any applications submitted but no call booked?

---

## Execution Block (30–60 min)

### Distributor
- Post content per schedule
- Send WhatsApp broadcast if scheduled
- Repurpose reel to story

### Scout
- Surface 10 new targets
- Add to tracking sheet with platform + handle

### Storyteller
- Generate today's content or DM batch
- Review hooks — test a new one if cadence allows

### Closer
- Respond to all inbound DMs
- Send follow-ups per sequence
- Qualify any new conversations

---

## Evening Block (10 min)

### Log results
Fill in daily metrics:
- Posts published
- DMs sent
- Replies received
- New qualified leads
- Applications submitted
- Calls booked

### Flag anything for Captain
- Unusual engagement on a piece
- Hot lead that needs attention
- Anything that broke or didn't work

---

## Non-Negotiables
- Never skip the morning assign
- Never let DMs sit unread for more than 4 hours
- Never post without a CTA
- Never end the day without logging results
