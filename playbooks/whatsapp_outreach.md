# WhatsApp Outreach Playbook

## Purpose
Use WhatsApp for warm outreach, broadcast messages to existing contacts, and follow-up sequences during active campaigns.

## Rules
- Only message people who know you or have engaged with your content
- Never send the same message to everyone — personalize the opener
- One message at a time — wait for a reply before the next
- Never pitch in the first message

---

## Broadcast Message Templates

### Broadcast 1: Campaign Announcement (Retreat)
```
Hey [Name] 👋

Just opened applications for the May retreat.

It's called Build While You Heal — 5 days for women who are winning but want the structure underneath to match.

If you've been thinking about systems, clarity, or just finally building the machine — this is it.

Applications close April 25. Only a few seats.

Want the link?
```

### Broadcast 2: Urgency Push (Final Week)
```
3 seats left for May.

If you've been watching and thinking — now's the time.

Build While You Heal.
5 days. Real work. Real system.

DM me or reply here and I'll send you the application.
```

## 1-on-1 Outreach Templates

### Opener (Cold-Warm — they know you but haven't engaged recently)
```
Hey [Name], been a while.

Saw you've been [building / posting / moving] — respect.

I'm running something in May for women at your level who want to build the system underneath the success.

Not sure if it's relevant for you — but wanted to put it on your radar.

Worth a quick convo?
```

### Opener (Direct — they've engaged with your content)
```
Hey — saw you liked/commented on [post].

That post was about [topic] — and it connects directly to what I'm building for May.

Are you dealing with [pain point] right now?
```

---

## Follow-Up Sequence (if no reply)

**Day 3:**
```
Just following up — didn't want this to get buried.

Is now a good time to talk?
```

**Day 7:**
```
Last thing I'll send on this.

If May doesn't work, no problem — I'll keep you in mind for the next one.

But if you're curious, here's the application: [LINK]
```

---

## Qualification Questions (once they reply)
1. "What does your business / work situation look like right now?"
2. "What's the main thing you feel like you're missing on the systems side?"
3. "Have you done anything like a retreat or intensive before?"

Use replies to qualify. If green → send application link. If yellow → nurture. If red → close warmly.
