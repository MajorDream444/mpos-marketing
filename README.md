# MPOS Marketing

Clawbot-powered marketing division for MAJOR PRIME OS.

## Mission
Generate attention, conversations, and revenue.

## Structure

```
YOU → MAJOR PRIME OS → CLAUDE → CLAWBOT AGENTS → OUTPUT
```

MPOS Marketing is not a content machine. It is a conversion machine. Every piece of content, every DM, every hook exists for one purpose: move qualified people toward application, call, or sale.

## Agent Roles

| Agent | Role |
|-------|------|
| Captain | Orchestrates campaigns, assigns tasks, tracks output |
| Scout | Finds leads, surfaces high-intent targets |
| Storyteller | Creates hooks, scripts, captions, story sequences |
| Distributor | Publishes and spreads content across platforms |
| Closer | Converts conversations into applications and sales |

## Active Campaigns

| Campaign | Status | Goal |
|----------|--------|------|
| Build While You Heal (May 2026) | Active | Fill retreat cohort |
| Authority Engine | Active | Position MPOS as the system |
| Viva La Contour | Queued | Sell first machine to med spa |

## Daily Workflow

1. Captain assigns daily campaign focus
2. Scout surfaces 10 high-fit targets
3. Storyteller generates 3 content pieces + 5 outreach messages
4. Distributor publishes and sends
5. Closer handles replies and moves leads to application or call
6. Captain logs results

## Metrics That Matter

- DMs sent
- Replies received
- Qualified leads identified
- Applications started
- Calls booked
- Sales closed

## Philosophy

- Prioritize buyers over viewers
- Prioritize conversations over content
- Prioritize revenue over vanity metrics
- Speed and execution over perfection
