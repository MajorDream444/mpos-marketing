# Skill: Lead Scout

## Purpose
Find high-fit leads for active campaigns.

## Inputs
- Campaign name
- ICP description
- Platform preference
- Geography (if relevant)
- Number of leads needed

## Process
1. Identify where the ICP spends time (IG, TikTok, LinkedIn, Google Maps)
2. Search using ICP signals (keywords, hashtags, follower patterns, bio keywords)
3. Filter by fit: income signals, activity, content alignment
4. Output top results only — no bloat

## Outputs
- Lead name / handle
- Platform
- Why they fit the ICP
- Signal that surfaced them
- Suggested opener angle

## Rules
- Top 10 per session maximum
- Flag high-priority leads separately
- Never include low-confidence leads to hit a number target

## Example Output Format
```
Lead: @handle
Platform: Instagram
Fit: Female creator, ~20k followers, sells digital product, posts about business + burnout
Signal: Active content, no system content, comments on MPOS-adjacent posts
Opener angle: Content burnout + system reframe
```

## Success Metric
- Leads surfaced per session
- Conversion rate from lead to DM reply
