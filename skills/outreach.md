# Skill: Outreach

## Purpose
Generate personalized outreach messages for cold-warm and warm leads.

## Inputs
- Lead type (ICP category)
- Platform (IG DM, WhatsApp, Email)
- Campaign / offer
- Any specific context about the lead

## Process
1. Reference something real about the lead (their work, content, or situation)
2. Create curiosity without pitching
3. End with one question
4. Prepare follow-up sequence in advance

## Outputs
- Opening message
- Follow-up message 1 (Day 3)
- Follow-up message 2 (Day 7)
- Loop-close message (Day 12)

## Rules
- No walls of text
- No copy-paste feel
- One idea per message
- Never pitch in message 1

## Success Metric
- Reply rate per outreach batch
- Conversations started per week
