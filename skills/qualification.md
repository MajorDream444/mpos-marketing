# Skill: Qualification

## Purpose
Assess leads and conversations — decide next step immediately.

## Inputs
- Conversation transcript or lead description
- Campaign / offer context

## Process
1. Check for income / business signal
2. Check for pain match
3. Check for urgency / readiness
4. Assign Green / Yellow / Red
5. Generate recommended next message

## Outputs
- Lead status (Green / Yellow / Red)
- Reason for status
- Recommended next message
- Any escalation flags

## Rules
- No grey area — every lead gets a status
- Green leads get application link within 24 hours
- Red leads get a clean close — no lingering
- Flag anything unusual to Captain immediately

## Success Metric
- Accuracy of qualification (Green leads that convert)
- Speed of qualification (hours from first reply to status assigned)
