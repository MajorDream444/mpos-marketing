# Skill: Conversion

## Purpose
Move qualified (Green) leads from conversation to application or call.

## Inputs
- Qualified lead (Green status confirmed)
- Campaign / offer
- Any specific objections raised

## Process
1. Confirm fit with one final question
2. Present next step clearly — application or call
3. Handle any objections with empathy + redirect
4. Set a clear deadline or urgency if relevant
5. Book call or confirm application submission

## Outputs
- Application delivery message
- Call booking message
- Objection response scripts
- Confirmation message after submission

## Common Objections

**"I need to think about it"**
→ "Totally fair. What's the main thing you're weighing?"

**"What's the price?"**
→ "The investment is discussed on the call — I want to make sure it's the right fit before we get there. Want to book 15 min?"

**"Is this right for me?"**
→ "Based on what you've shared — yes. Here's why: [specific reason]. The application takes 3 minutes. Want to start there?"

## Success Metric
- Applications submitted per week
- Calls booked per week
- Conversion rate from Green lead to application
