# Skill: Reporting

## Purpose
Generate daily and weekly performance reports for MPOS Marketing.

## Inputs
- Daily metrics log
- Weekly metrics sheet
- Campaign status

## Daily Report Format
```
MPOS MARKETING — DAILY REPORT
Date: [date]
Campaign: [active campaign]

OUTPUT
Posts published: [#]
DMs sent: [#]
Replies received: [#]

PIPELINE
New qualified leads: [#]
Applications sent: [#]
Applications submitted: [#]
Calls booked: [#]

FLAGS
[Any unusual activity, hot leads, or issues]

TOMORROW
[Captain's assignment for the next day]
```

## Weekly Report Format
```
MPOS MARKETING — WEEKLY REPORT
Week of: [date range]
Campaign: [active campaign]

SUMMARY
Total posts: [#]
Total DMs sent: [#]
Total conversations: [#]
Total qualified leads: [#]
Total applications: [#]
Total calls: [#]
Total sales: [#]

BEST PERFORMING CONTENT
[Title or hook of top piece]
[Why it worked]

WHAT TO CHANGE
[One thing to adjust next week]

WHAT TO DOUBLE
[One thing that's working — do more]

NEXT WEEK PRIORITY
[Top campaign focus for next 7 days]
```

## Rules
- Daily report generated end of day
- Weekly report generated Sunday evening
- Flag any anomalies — positive or negative
- Never skip reporting even on low-activity days

## Success Metric
- Reporting consistency
- Data accuracy
- Actionable insights per report
